# 🎬 Filmoteca

**Filmoteca** es un sistema de gestión de películas y series desarrollado en **Java con Spring Boot**.  
El proyecto permite organizar, consultar y administrar información relacionada con películas, series, géneros, directores y usuarios.  

## 🚀 Tecnologías utilizadas
- Java 17
- Spring Boot 3
- Spring Data JPA
- Swagger / OpenAPI (documentación de la API)

## 📂 Módulos principales
- **Contenido** → Películas, Series, Géneros, Directores.  
- **Usuarios** → Usuarios, Roles, Perfiles.  
- **Interacciones** → Reseñas, Listas de reproducción.
